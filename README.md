 ## DocuChat Pro
# Installation

1.  Clone the repository
    ```bash
    git clone [https://github.com/YOUR_USERNAME/DocuChat-Pro.git](https://github.com/YOUR_USERNAME/DocuChat-Pro.git)
    cd DocuChat-Pro
    ```

2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the App**
    ```bash
    streamlit run app.py
    ```

# How to Use
1.  Launch the app.
2.  Select your preferred brain (**Gemini 3 Pro** or **GPT-4o**) in the sidebar.
3.  Paste your API Key (Google AI Studio or OpenAI).
4.  Upload a PDF document.
5.  Ask  questions

##  Requirements
* Python 3.10+
* A Google Cloud API Key  or OpenAI API Key.
